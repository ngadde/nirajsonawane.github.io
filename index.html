Java Process Builder

The ProcessBuilder class is used to create separate operating system processes. There are many scenarios, Where we need to launch separate operating system processes form java program.

Before JDK 5.0, We need to use the exec() method of the java.lang.Runtime class to start new process.

JDK 5.0 has added ProcessBuilder to Start new OS process.

Note:

ProcessBuilder class is not synchronized. If multiple threads access a ProcessBuilder instance concurrently, and at least one of the threads modifies one of the attributes structurally, it must be synchronized externally.
